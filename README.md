# saytouche
프로젝트 개요

Saytouche 공식 쇼핑몰을 참고해

메인 페이지를 포함한 다중 페이지 구조의 웹사이트를 직접 디자인하고 퍼블리싱한 프로젝트입니다.



본 프로젝트에서는 해당 구조와 시각 흐름을 참고하되

단순 복제가 아닌 레이아웃 구성, 이미지 배치, 브랜딩 톤을 재해석해 자체 디자인으로 구현했습니다.



메인 페이지뿐 아니라 서브 페이지 전반에 걸쳐

페이지별 콘텐츠 성격에 맞는 레이아웃과 비주얼 흐름을 설계했으며,

브랜드의 트렌디하고 감각적인 이미지를 강조하는 데 중점을 두고 제작했습니다.



퍼블리싱 관점에서 페이지 비주얼 흐름, 시맨틱 마크업, CSS 레이아웃 구성 능력을 적용했습니다.



실현 기능 (Coding 중심)

	•	메인 페이지 및 서브 페이지를 포함한 다중 페이지 구조 구현

	•	페이지별 콘텐츠 성격에 맞춘 레이아웃 및 마크업 분리

	•	브랜드 이미지를 강조하기 위한 이미지 중심 레이아웃 구성

	•	CSS를 활용한 여백, 정렬, 비율 조절로 시각적 흐름 제어

	•	간단한 모션·움직임 요소를 적용해 트렌디한 브랜드 이미지 표현

	•	공통 UI 요소를 클래스 단위로 관리해 코드 재사용성 확보

	•	시맨틱 태그를 활용한 구조적인 마크업 작성
